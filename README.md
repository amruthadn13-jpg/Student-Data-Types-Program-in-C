# Student Data Types Program in C

## Overview

This project is a simple C program that demonstrates the use of basic data types, variables, and formatted output.

The program stores and displays:

* Student ID
* Total Score
* Final Grade

It also demonstrates the use of:

* Integer data type
* Float data type
* String handling in C
* Formatted printing using `printf()`

---

## Concepts Covered

* Variables in C
* Data types:

  * `int`
  * `float`
  * `char array (string)`
* Output formatting
* String printing using `%s`

---

## Project Description

The program stores student information using different data types and prints the details in a formatted manner.

### Stored Information

* Student ID
* Score
* Grade (`A+`)

The grade is stored as a string because multiple characters cannot be stored in a single `char` variable.

---


## Key Concepts Used

### Integer

```text id="tq4jzz"
int studentID = 05;
```

Stores whole numbers.

### Float

```text id="y09xho"
float score = 87.5;
```

Stores decimal values.

### String

```text id="n1pb55"
char grade[] = "A+";
```

Stores multiple characters as a string.
